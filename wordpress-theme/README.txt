Professional security company one-page WordPress theme.

== Description ==

Security One is a professional, responsive WordPress theme designed specifically for security companies. This one-page theme includes:

* Responsive design that works on all devices
* Professional hero section with customizable content
* Services showcase section
* About section highlighting company credentials
* Contact form with email integration
* Mobile-friendly navigation
* Customizer integration for easy content management
* Security-focused color scheme and design

== Installation ==

1. Upload the theme files to the `/wp-content/themes/security-one/` directory
2. Activate the theme through the 'Appearance > Themes' menu in WordPress
3. Customize your site using the WordPress Customizer (Appearance > Customize)
4. Set up your navigation menu in Appearance > Menus
5. Configure contact information in the Customizer

== Customization ==

This theme includes several customizer options:

* Hero section title and subtitle
* Contact information (phone, email, address)
* Site logo and branding
